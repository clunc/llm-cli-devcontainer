# llm-cli-devcontainer   VS Code devcontainer template with Node 20, pnpm, Git, GitHub CLI, Codex CLI, Claude CLI, and extensions preinstalled. Copy `.devcontainer/` into a repo, adjust   ports/postCreate/postAttach commands, reopen in container, and start coding with LLM CLIs ready. Ports include 1455 for Codex auth.
